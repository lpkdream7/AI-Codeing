# 版本记录

## v0.3.1 — 2026-07-22 · HTTP 白屏修复

### 修复

- 在尚未启用 HTTPS 时关闭 CSP 的 `upgrade-insecure-requests`，避免浏览器把前端静态资源强制升级到 HTTPS 后出现白屏
- 保留 HTTPS 模式下的资源升级策略；域名启用 HTTPS 后仍使用完整安全策略

## v0.3.0 — 2026-07-22 · Docker 自托管

- 新增 Node.js 自托管服务，并复用现有待办界面
- 新增邮箱注册、密码登录、退出和服务端会话
- 接入 MySQL 8.4，支持任务增删改查、旧数据导入和多端冲突检测
- 增加 Dockerfile，以及应用、MySQL、Caddy 三服务 Compose 编排
- 数据库和应用仅通过 Docker 内网通信，MySQL 不映射公网端口
- 密码使用 Argon2id，原始会话令牌不写入数据库
- 增加每天北京时间 05:00 的 mysqldump 备份，默认保留 30 天
- 增加 Ubuntu 24.04、Docker Compose、MySQL 8.4 自托管环境说明
- 增加 MySQL 初始化 SQL，覆盖用户注册、登录会话、邮箱验证、密码重置和待办任务
- 增加一键生成数据库密码并启动完整服务的部署脚本

## v0.2.0 — 2026-07-22 · 云端同步

- 使用 ChatGPT 登录身份建立个人账户
- 使用 D1 数据库保存用户和任务
- 支持同一账户在手机、平板和电脑间同步
- 首次登录时自动迁移浏览器中的旧任务
- 增加后台轮询、窗口重新聚焦同步和同步状态提示
- 使用任务版本号检测多端并发修改，避免静默覆盖
- 所有任务接口均执行服务端身份和数据归属检查

## v0.1.0 — 首个可用版本

- 新增、编辑、完成和删除任务
- 支持筛选、搜索、优先级、截止日期与进度统计
- 支持响应式桌面和手机布局
