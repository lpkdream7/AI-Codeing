# 版本记录

## 未发布 — 自托管准备

- 增加 Ubuntu 22.04、Docker Compose、MySQL 8.4 自托管环境说明
- 增加 MySQL 初始化 SQL，覆盖用户注册、登录会话、邮箱验证、密码重置和待办任务
- 增加自托管环境变量示例，且不包含真实凭据

## v0.2.0 — 2026-07-22 · 云端同步

- 使用 ChatGPT 登录身份建立个人账户
- 使用 D1 数据库保存用户和任务
- 支持同一账户在手机、平板和电脑间同步
- 首次登录时自动迁移浏览器中的旧任务
- 增加后台轮询、窗口重新聚焦同步和同步状态提示
- 使用任务版本号检测多端并发修改，避免静默覆盖
- 所有任务接口均执行服务端身份和数据归属检查

## v0.1.0 — 首个可用版本

- 新增、编辑、完成和删除任务
- 支持筛选、搜索、优先级、截止日期与进度统计
- 支持响应式桌面和手机布局
